Amazon Sales Report Dashboard

Overview

This project presents a comprehensive Amazon Sales Report Dashboard designed to analyze sales performance and customer behavior using data-driven insights. The dashboard helps stakeholders understand overall revenue trends, order patterns, product performance, and operational efficiency through interactive visualizations.
Objectives
The primary objective of this report is to track key business metrics such as total revenue, total orders, average order value, customer purchase frequency, and order status distribution. It also aims to identify high-performing products, profitable categories, and payment method preferences to support better decision-making.

Key Features

Sales performance analysis by time period
Customer insights including repeat vs one-time customers
Revenue and order breakdown by product and category
Order status analysis (Delivered, Shipped, Cancelled, Returned)
Payment method distribution (UPI, Card, COD)
Seller and operational performance overview

Tools & Technologies

Power BI for data modeling and visualization
Excel / CSV as the data source
DAX for calculated measures and KPIs

Insights & Use Case

This dashboard enables businesses to monitor growth, optimize inventory, improve customer retention, and identify potential refund or cancellation risks. It serves as a practical analytics solution for e-commerce sales performance tracking.
